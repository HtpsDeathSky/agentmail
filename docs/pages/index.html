MVP Capabilities

Mail first, AI by deliberate action.

Windows desktop shell

Tauri v2 app with a Rust backend, React/Vite UI, and SQLite local state.

IMAP/SMTP workflow

Manual account setup, connection tests, folder sync, search, reading, and direct send.

Local mail database

Accounts, folders, messages, sync state, search index, audits, and AI insights live locally.

Manual remote AI

Selected messages can be analyzed through an HTTPS OpenAI-compatible provider.

Current Limits

Clear boundaries for the current build.

  • Mailbox passwords and AI API keys are stored plaintext in SQLite for this MVP.
  • AI analysis is manual only; incoming mail is not analyzed automatically.
  • OAuth, tray mode, notifications, attachment downloads, and rules are not implemented yet.
  • Use provider app passwords where required by the mail provider.
